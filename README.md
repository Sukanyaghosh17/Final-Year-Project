# FIR Automation System

This project automates FIR generation and analysis using AI/ML.

## Project Structure

- **backend/**: Flask application for the API and ML services.
- **frontend/**: React/Vite application for the user interface.
- **research/**: Contains research notebooks, datasets, and analysis.
  - **crime_analysis/**: Crime data analysis.
  - **crime_prediction/**: Crime prediction models.
  - **fir_research/**: FIR automation research.
- **docs/**: Project documentation and presentations.
- **scripts/**: Utility scripts to start the application.

## Getting Started

### Prerequisites

- Python 3.10+
- Node.js 16+

### Running the Application

1. **Backend**:

   ```bash
   scripts\start_backend.bat
   ```

2. **Frontend**:
   ```bash
   scripts\start_frontend.bat
   ```
